# finops.apis.io — FinOps Profiles

[**finops.apis.io**](https://finops.apis.io) is the finops profiles slice of the [APIs.io](https://apis.io) network.

**Live site:** [https://finops.apis.io](https://finops.apis.io)

## What's in this repo

- **`_finops/`** — one Markdown file per provider, generated from upstream `finops/` directories in the [api-evangelist](https://github.com/api-evangelist) repos.
- **`index.html`, `_layouts/`, `_includes/`, `assets/`** — Jekyll site chrome
- **`_config.yml`** — Jekyll site configuration

## How it's built

This collection is regenerated by [`network/scripts/build.py`](https://github.com/api-search/network/blob/main/scripts/build.py), which reads each provider's `finops/*.yml` files from the upstream [api-evangelist](https://github.com/api-evangelist) repos and emits records here.
